# PiVPN Docker
A simple docker container that sets up pivpn.

# Install and Config
Type in:
```
curl -s -L https://bit.ly/2CzLV3T  -o install.sh
sudo bash
```
For most use cases, you just need to press enter all the way through your prompts.
